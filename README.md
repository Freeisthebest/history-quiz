# 天津大学近代史选择题刷题网站

面向《中国近现代史纲要》选择题复习的纯静态刷题网站。无需登录、不用填写考号或班级，打开后即可选择练习方式。

在线地址：https://freeisthebest.github.io/history-quiz/

## 主要功能

- 首页选择练习方式：模拟试卷、即时刷题、错题重练，也可进入题库、收藏栏和更新日志。
- 模拟试卷：每次随机抽取 40 道单选题和 20 道多选题，整套答完后统一结算；未作答题目会单独列出，且不会进入错题库。
- 即时刷题：完整题库洗牌后逐题练习，一轮抽完前不会重复；单选题选择后立即判定，多选题点击“提交本题”后判定；手动重新洗题前会再次确认。
- 错题记录：答错且已作答的题目会进入浏览器本地错题库，可查看历史错题；错题重练沿用即时反馈，答对后可选择移出错题库。
- 收藏栏：任意题目可标记收藏，便于后续集中复习或核对。
- 查看题库：可浏览全部题目、正确答案和本次作答情况，支持单选/多选筛选和关键词搜索。
- 更新日志：网站内读取 `CHANGELOG.md`，用于同步展示最近改动。

## 本地文件

- `index.html`：网页入口，也包含静态资源版本号。
- `styles.css`：页面样式。
- `app.js`：页面交互、视图切换、错题/收藏/题库展示逻辑。
- `quiz-core.js`：组卷、判分、错题同步、洗题等核心逻辑。
- `questions.js`：题库数据，新增或修改题目主要编辑这个文件。
- `CHANGELOG.md`：网站内同步展示的更新日志。

## 题库维护

题目统一写在 `questions.js` 的 `QUESTION_BANK` 中。每道题建议保持下面结构：

```js
{
  "id": "q001",
  "sourceNo": "1",
  "type": "single",
  "question": "题干内容",
  "options": {
    "A": "选项 A",
    "B": "选项 B",
    "C": "选项 C",
    "D": "选项 D"
  },
  "answer": ["A"],
  "explanation": ""
}
```

维护时注意：

- `id` 不要重复；已发布题目尽量不要改 `id`，否则会影响用户本地错题和收藏记录的对应关系。
- `type` 填 `"single"` 或 `"multiple"`。
- `answer` 必须是数组；单选题只放一个字母，多选题放多个字母。
- 修改题干、选项或答案后，建议同步更新 `CHANGELOG.md`。
- 如果修改了 `app.js`、`quiz-core.js` 或 `questions.js`，建议同步更新 `index.html` 中对应资源链接后的 `?v=...` 版本号，以减少旧浏览器缓存影响。

## 本地预览

直接双击 `index.html` 可以预览主要功能。如果更新日志没有显示，可以在本目录启动静态服务器：

```powershell
python -m http.server 58325
```

然后打开：

```text
http://127.0.0.1:58325/index.html
```

## 发布

本项目通过 GitHub Pages 发布。确认本地预览无误后，提交并推送到 `main` 分支即可，GitHub Pages 通常会在 1-2 分钟内更新线上网站。

## 数据说明

用户的做题记录、错题库和收藏保存在各自浏览器的 `localStorage` 中，不会上传到服务器。更新题库不会主动清空这些记录；网站会尽量用当前题库的最新题干、选项和答案显示旧错题或收藏。
